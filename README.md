# 医療費 CSV → Excel 変換ツール

マイナポータルから取得した医療費 CSV を、国税庁配布の `iryouhi_form_v3.1.xlsx` に貼り付け可能なタブ区切り形式へ変換するツールです。

## 前提条件

- Node.js（v18 以上）

## 国税庁フォームについて

`template/iryouhi_form_v3.1.xlsx` は、国税庁が公開している「医療費集計フォーム」です。  
フォームは更新される可能性があるため、使用前に必ず国税庁の公式サイトから最新版を確認してください。

- 配布ページ：https://www.nta.go.jp/taxes/shiraberu/shinkoku/tokushu/keisubetsu/iryou-shuukei.htm

最新版のファイル名が変更されている場合は、ダウンロードしたファイルを `template/` フォルダ内のものと差し替えてください。

## 使い方

変換スクリプトを、ダウンロードした CSV と同じフォルダに置き、次を実行します：

```sh
npx tsx index.ts
```

カレントディレクトリ内のすべての `.csv` ファイルが自動的に処理されます。

### 特定のファイルのみ処理したい場合

```sh
npx tsx index.ts "医療費情報A_20260304202012215.csv" "医療費情報B_*.csv"
```

### 出力をファイルに保存したい場合

```sh
npx tsx index.ts > output.tsv
```

## Excel への貼り付け手順

1. `template/iryouhi_form_v3.1.xlsx` を開く  
2. シート `医療費集計フォーム` の **B9** セルを選択  
3. 変換結果を貼り付け（Ctrl+V）

列 B〜I が自動で入力されます。列 A（No）は既存の数式により自動計算されます。

## 列の対応関係

| Excel列 | 見出し | 元データの項目 |
|---|---|---|
| A | No | 自動生成（`=ROW()-8`） |
| B | 医療を受けた人 | CSV ヘッダー `氏名` |
| C | 病院・薬局などの名称 | `医療機関等名称` |
| D | 診療・治療 | 医科外来 または 歯科外来 の場合に `該当する` |
| E | 医薬品購入 | 調剤 の場合に `該当する` |
| F | 介護保険サービス | （空欄） |
| G | その他の医療費 | （空欄） |
| H | 支払った医療費の金額 | `窓口相当負担額（円）` |
| I | 左のうち、補填される金額 | `0` |
| J | 支払年月日 | `診療年月` を `YYYY/MM` 形式に変換 |

## 注意事項

- UTF-8 と cp932（Shift-JIS）の CSV を自動判別して読み込みます。  
- 引数なしで実行した場合、カレントディレクトリ内のすべての `.csv` が対象になります。重複ファイルやテスト用 CSV があると二重計上されるため、不要なファイルは削除してください。  
- 列 J は `YYYY/MM`（例：`2025/02`）形式です。元 CSV に日付（年月日）がなく、年月のみのためです。  
- 列 I（補填される金額）はデフォルトで `0` です。高額療養費や保険金などで補填がある場合は、Excel 上で手動で修正してください。
